# Что такое параметр функции?

Функция может использовать эту информацию для выполнения вычислений 
или для принятия решения о том, какую из множества возможных инструкций 
следует выполнить.

ВАЖНО!
Как мы знаем функция с параметрами имеет return. 
return может показывать что происходит с параметрами (/ * + - ), 
или может возвращать переменную в которую уже поместили всё необходимое:

function valueNumbers(firstNum, secondNum) {
  let summNumbers = (firstNum + secondNum) * 2
  return summNumbers
}
valueNumbers(10, 15)

чтобы проверить результат вызываем функцию в console.log();
console.log(valueNumbers(10, 15));

8)
NaN (Not-A-Number)
специальное значение "не число", которое обычно говорит о том, 
что была выполнена бессмысленная операция.
Например: console.log(5 * 'q');
5 - number
'q' - string
бессмысленно умножать 5 на q, получаем математическую ошибку (NaN)

Функция `isNumber()` проверяет, является ли входная переменная числом.
Подробнее тут: https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/isNaN

Задача: дана функция

function isValidNumber(num) {

}
Устройте проверку, передавая параметры в виде цифр или букв.
Например:

isValidNumber(11)
Получаем в консоле: 11 is a valid number

но если такой параметр:
isValidNumber("www")
Получаем в консоле: www is not a number
